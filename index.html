***ĐƯỜNG ĐI TRONG WEB DEV***

Lộ trình 2 năm học fullstack

  1. Frontend Cơ Bản (2-3 tháng)

    Mục tiêu: Nắm vững HTML, CSS, JavaScript cơ bản.

    • HTML & CSS: Học từ cơ bản đến nâng cao, bao gồm:
      • Semantic HTML, Forms, Tables, Flexbox, Grid.
      • Responsive Web Design, CSS Animation.
    • JavaScript cơ bản:
      • Biến, kiểu dữ liệu, toán tử, vòng lặp, hàm.
      • DOM, Event Listener, LocalStorage.
      • Callback, Promise, Async/Await.
    • Dự án: Xây dựng trang web tĩnh, ứng dụng JavaScript nhỏ.
    • Nguồn học: Kênh Gà Lại Lập Trình trên YouTube.
      • LINK YOUTUBE

  2. Frontend Nâng Cao (2-3 tháng)

    Mục tiêu: Làm việc với thư viện, framework.

    • CSS nâng cao:
      • SCSS, Tailwind CSS.
    • JavaScript nâng cao:
      • ES6+, Fetch API, Error Handling.
    • Thư viện & Framework:
      • React.js (Component, Props, State, Hook, Router).
    • Nguồn học: Tiếp tục học từ Gà Lại Lập Trình, kết hợp với tài liệu khác.

  3. Backend Cơ Bản (2-3 tháng)

    Mục tiêu: hiểu cách xây dựng server và API

    • Node.js & Express.js:
      • Routing, Middleware, REST API.
    • Database:
      • MongoDB hoặc PostgreSQL.
      • CRUD (Create, Read, Update, Delete).
    • Xác thực & Bảo mật:
      • JWT, bcrypt.
    • Dự án: Xây dựng API cho ứng dụng web.

  4. Fullstack (3-4 tháng)

    Mục tiêu: Kết hợp Frontend & Backend để tạo web hoàn chỉnh.

    • Kết nối React với Backend.
    • State Management (Redux hoặc Context API).
    • Deploy ứng dụng web lên Vercel/Render và Railway.
    • Dự án lớn: Xây dựng website thực tế.

  5. Học nâng cao & Thực tập (6 tháng)

    • Học TypeScript, Next.js.
    • Học CI/CD, Docker, Microservices (cơ bản).
    • Làm dự án cá nhân & tham gia open-source.
    • Ứng tuyển thực tập & làm portfolio.
